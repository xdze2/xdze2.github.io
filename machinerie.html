Machinerie

mars 2017 à propos

Ce site web est généré avec le moteur de site html statique Pelican, avec un thème perso (blue robot). Le plugin Liquid Tags permet d'intégrer directement les notebooks Jupyter-Python3 comme articles.

Le code Markdown est utilisé pour mettre en forme les textes, et Latex (symbols list) pour les équations (à travers la librairie javascript mathJax).

Le site est hébergé sur Github, et le developpement suivit avec Git.
