Méthode des moindres carrés¶

La méthode la plus célèbre est biensûr la méthode des moindres carrés. On recherche la droite $y^\star(x)$ tel que:

$$ y^\star - \bar y = a \, ( x - \bar x ) $$

$\bar x$, $\bar y$ sont les moyennes.

La pente $a$ est obtenue par la formule : $$ a = \frac{ \Sigma (y_i - \bar y) (x_i - \bar x) }{ \Sigma (x_i - \bar x)^2 } $$

def moindrescarres(X, Y):
    xm, ym = np.mean( X ), np.mean( Y )
    
    a = np.sum( (Y-ym)*(X-xm) )/np.sum( (X-xm)**2 )
    b = ym - a*xm
    
    return a, b

from scipy import optimize

def regressionL(X, Y):
    costfun = lambda P: np.sum( np.abs( Y - (P[0]*X+P[1]) ) )
    Pzero = moindrescarres(X, Y)
    r = optimize.minimize(costfun, Pzero)
    return r.x
